Le plugin loomup vous permez de retrouvez vos vidéos loomup, accompagnées chacune
d'un shortcode pour pouvoir les intégrer facilement dans vos pages wordpress.

== Description ==

LOOMUP 1° solution européenne de video connectée au service de la relation client.

LOOMUP **RÉINVENTE** LA MANIÈRE DE *S’ADRESSER AUX CLIENTS*
Une vidéo personnalisée est unique pour le client qui la visionne.
Elle se compose d’un socle commun qui raconte une histoire déterminée par l’annonceur, et s’enrichit d’éléments propres à chaque destinataire, en s’appuyant sur les données CRM.
Le consommateur peut visionner sa vidéo personnalisée autant de fois qu’il le souhaite au fil des mois, sur tous les supports (ordinateur, tablette, smartphone).
Ses données sont mises à jour en temps réel, au moment de la lecture.



1. La version beta vous permet de tester une démo
1. La vidéo de démonstration présente le récapitulatif d'une commande d'un client
1. La vidéo se personnalise en fonction des données du client

== Installation ==

1. Téléchargez puis placez le plugin sous `/wp-content/plugins/loomup` , ou installez le plugin directement depuis l'interface Wordpress.
1. Activez le plugin depuis la page Plugins
1. Configurez le plugin en cliquant sur `Loomup` dans la barre latérale
1. Enregistrez  votre clé loomup.
